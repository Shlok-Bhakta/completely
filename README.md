# Completely Code Completion 

This project is not ready for anyone yet. I am still hard at work figuring out what I want to do and where I want to go and how I am gonna get the compute for this.

This is a crazy project to try and make a complete language model specifically for code completion. This will also come with a whole suite of tools to use in day to day coding. Ideally this is a lightweight, local, and secure version of tools like tabnine, supermave, etc.
